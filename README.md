
# ChainView 웹 관리자 UI

ChainView 웹 관리자 UI입니다. 서버, 서비스, 서비스 관계, 장애 영향도를 관리하는 운영 포털 화면으로 구성되어 있습니다.

Original Figma project:
https://www.figma.com/design/wk5cHZqaUPUEHUXq7pirez/ChainView-%EC%9B%B9-%EA%B4%80%EB%A6%AC%EC%9E%90-UI

## Requirements

Node.js 20 이상을 권장합니다. `react-router@7.13.0`이 Node 20 이상을 요구합니다.

## Setup

```bash
git clone https://github.com/cheal3/2026-AX-Squad-chainview.git
cd 2026-AX-Squad-chainview
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
```
